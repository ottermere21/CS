### Data Structure가 중요한 이유
- 디지털 세계는 매년 점점 많은 데이터를 처리하게 된다. 현재 존재하는 데이터의 대부분은 불과 최근 몇 년 사이에 생성된 것이다. 
- 약 0.0025ZB/일(2018, Forbes 기사 추정치) → 약 0.4ZB/일(2024, IDC Global DataSphere 연간 147 ZB를 일일로 환산) (제타 바이트)
- Data Structures는 대량 데이터를 관리하기 위해 필요하며, 알고리즘의 효율성을 높이는 핵심 요소
